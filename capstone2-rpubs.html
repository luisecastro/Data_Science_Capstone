Word predictor app

Main characteristics:

  • Optimized user interface, seamless navigation.
  • Immediate data load possible due to processing 583MB of raw data into 16MB.
  • Sweetspot of speed & accurancy with up to 3grams for prediction.
  • Backoff algorithm for best results on user input.
  • Easy user customization for displaying results.
  • Automatic as you type prediction.


For more detailed information visit: Predictor app link

Word predictor app

Bonus characteristics:

  • Corrector uses Levenshtein distance to provide correction and/or prediction to the currently typed word.
  • Phrase generator additional feature for the user, it allows to create phrases and selecting their length and randomness.
  • Shakespearean allows the user to switch back and forth from current English to old fancy English.
  • No external libraries, easier to update and patch without depending on 3rd parties, having full access to code and documentation.
